# Talking Points

## 30-Second Setup

ClawMax is a multi-agent orchestration platform for teams of AI agents, with templates, workflows, skills, groups, and execution visibility built in.

For this hack, we added a multimodal workflow called **Real-Time Research Desk**. The key idea is simple:
- ClawMax orchestrates the agents and workflow
- Senso is the shared multimodal memory and evidence layer
- agents ingest new evidence, retrieve prior context, and produce an action-ready brief

## What To Show

1. Show the applied org template and the agent team
2. Show the workflow DAG with kickoff, intake, retrieval, and brief
3. Show the evidence path and screenshots
4. Run the workflow live
5. Show the final brief artifact

## Key Points To Say During Demo

- These agents do not just chat. They act through a workflow.
- Senso gives the team shared memory, so agents can build on prior evidence instead of starting from zero.
- We added ClawMax-specific Senso adapter skills to reduce friction around ingest, retrieval, and handoff.
- The output is not just analysis. It is a concise action brief with evidence, confidence, and next step.

## If You Need One Sentence

ClawMax turns multimodal evidence into coordinated agent action, with Senso as shared context and workflows as the execution engine.
