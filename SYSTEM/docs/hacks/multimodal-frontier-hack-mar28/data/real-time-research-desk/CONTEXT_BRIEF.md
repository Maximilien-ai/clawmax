# Context Brief

This data pack is for the hackathon submission path.

## Goal

Demonstrate a Senso-backed multi-agent workflow that:

- ingests multimodal evidence
- retrieves grounded prior context
- synthesizes a concise brief
- recommends a clear next action

## What To Include In Evidence

- workflow DAG views
- apply-template flow
- any blocker/error screenshots that show agent or workflow friction
- visible Senso/skills configuration where relevant

## Expected Final Output

A short decision brief containing:

- summary
- top evidence
- confidence
- recommended next action
