# ClawMax Feature Themes

**Last Updated**: 2026-03-23
**Status**: Current summary

This file replaces the old March 15 feature list with the active themes that still matter.

## Current Feature Tracks

- **Workflow scale UX**
  - workflow table view
  - bulk actions and selection patterns matching agents/templates/communication
- **Agent coordination**
  - forward 1:1 chat to group
  - community rules and constraints inherited by groups
  - better blocked/waiting notifications across agents
- **Template system**
  - richer template categories and curated packs
  - export/import and sharing
  - stronger template apply guarantees
- **Cost and operations**
  - pause/disable agents
  - per-agent cost controls
  - workflow cron reliability
- **BYOK and auth**
  - secure per-user key storage after the preview flow
  - clean-room OAuth verification
