# RAG To Sandbox Launch Team

Use this template when you want agents to define a RAG system, pressure-test the retrieval strategy, and package the result for sandbox deployment with shared memory defaults.

Best for:
- RAG demos that need a deploy path
- doc or support assistants moving toward production
- combined Blaxel + Redis partner demos

Recommended integrations:
- Blaxel enabled in Workspaces Integrations
- Redis enabled in Workspaces Integrations
- GitHub if you want code and issue coordination

Typical flow:
1. Kick off the RAG problem and launch bar
2. Build the ingestion and indexing plan
3. Define retrieval and evaluation
4. Produce the sandbox launch plan
