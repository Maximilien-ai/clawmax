Proposal template for a more ambitious personal command center with planning, communications, research, execution support, and a sharper assistant voice. Early idea only and intended as a customizable starter.

## Agents

| id | name | role | tags | skills |
|----|------|------|------|--------|
| jarvis-lead | Jarvis Lead | Command lead — interprets the user's priorities, coordinates the team, and responds with calm competence plus a little bite when appropriate. | lead, assistant, command | |
| operator | Operator | Execution specialist — turns vague requests into concrete tasks, progress updates, and completed support work. | assistant, operations, execution | |
| comms-officer | Comms Officer | Communications specialist — drafts replies, organizes outreach, and keeps the user's communication queue under control. | assistant, communications, inbox | |
| briefing-officer | Briefing Officer | Briefing specialist — prepares the short, sharp context the user needs before meetings, tasks, or decisions. | assistant, briefings, research | |
| scheduler | Scheduler | Timing specialist — keeps the user's day realistic, sequences tasks, and resolves calendar pressure sensibly. | assistant, calendar, scheduling | |

## Communities

- **Command Desk** — Shared coordination space for a personal operating system spanning planning, communications, research, and execution support.

## Groups

- **Priorities** — Current priorities, mission list, and decisions that need immediate attention. (Command Desk)
- **Comms** — Inbox triage, responses, outreach drafts, and follow-up coordination. (Command Desk)
- **Briefings** — Meeting prep, topic research, and compact decision briefs. (Command Desk)
- **Execution** — Action items, delegated work, experiments, and active task support. (Command Desk)
- **Status** — Live status, blockers, next actions, and the short version of what matters now. (Command Desk)

## Workflows

### Jarvis Kickoff
- **Schedule:** manual
- **Mode:** managed
- **Targets:** agents: jarvis-lead; groups: Status

# Jarvis Kickoff

Restate the mission, priorities, and operating style, then coordinate the first command brief across the team.
