# Web Research & Synthesis Workflow

Test advanced web browsing, research, and information synthesis capabilities.

## Objective

Research a technical topic across multiple sources and synthesize findings into actionable insights.

## Part 1: Multi-Source Research

**Topic**: "Latest developments in AI agent frameworks (2025-2026)"

**Task 1.1**: Primary research:
- Search for "AI agent frameworks 2025"
- Identify top 3-5 frameworks mentioned
- Note key features of each

**Task 1.2**: Documentation dive:
- Pick one framework (e.g., LangChain, AutoGPT, OpenClaw)
- Find its official documentation
- Extract:
  - Core architecture
  - Main use cases
  - Latest version/release date

**Task 1.3**: Community insights:
- Search for recent blog posts or articles (2025-2026)
- Find 2-3 developer perspectives
- Note common pain points or praise

## Part 2: Comparative Analysis

**Task 2.1**: Feature comparison:
Create a comparison table of at least 3 frameworks covering:
- Multi-agent support
- Tool integration
- Memory/persistence
- Scheduling capabilities
- Community size

**Task 2.2**: Use case mapping:
- Identify which framework is best for:
  - Enterprise workflow automation
  - Research and analysis
  - Customer service automation
  - Development assistance

## Part 3: Trend Analysis

**Task 3.1**: Emerging patterns:
Based on your research, identify:
- 3 emerging trends in AI agent development
- 2 common challenges developers face
- 1 prediction for the next 6 months

**Task 3.2**: Technology stack:
- What programming languages dominate?
- Which LLM providers are most common?
- What infrastructure requirements are typical?

## Part 4: File Operations & Data Storage

**Task 4.1**: Save research notes:
Create a file `~/research-findings.md` containing:
```markdown
# AI Agent Frameworks Research

## Date: [Current date]

## Frameworks Analyzed
1. [Framework 1]
   - Features: ...
   - Strengths: ...
   - Use cases: ...

2. [Framework 2]
   ...

## Key Trends
- [Trend 1]
- [Trend 2]

## Recommendations
...
```

**Task 4.2**: Data extraction:
- Extract URLs of the top 5 sources you used
- Save them to a file called `sources.txt`
- Include brief annotations for each

## Part 5: Synthesis & Recommendations

**Task 5.1**: Executive summary:
Write a 4-6 sentence summary covering:
- State of AI agent frameworks in 2026
- Most mature vs most innovative options
- Who should use what

**Task 5.2**: Recommendations:
For someone building a multi-agent system today, recommend:
- Best framework for their needs
- Key considerations
- Potential pitfalls to avoid

## Part 6: Verification & Quality Check

**Task 6.1**: Fact checking:
- Verify at least 2 claims you made with additional sources
- Note any conflicting information found
- Rate your confidence in your findings (1-10)

**Task 6.2**: Source quality:
- Rate the quality of your sources:
  - Official documentation
  - Blog posts
  - Community forums
- Note any gaps in available information

## Output Format

Provide your complete research report including:
1. Executive summary
2. Detailed findings (frameworks, trends, stack)
3. Comparative analysis
4. Recommendations
5. Source citations with annotations
6. Confidence assessment

## Success Criteria

- ✅ Research from 5+ sources
- ✅ Create comparison of 3+ frameworks
- ✅ Identify clear trends
- ✅ Provide actionable recommendations
- ✅ Save findings to files
- ✅ Cite all sources

## Notes

- If web browsing is limited, use available search APIs
- Focus on quality over quantity
- Be honest about confidence levels
- Note any paywalls or access restrictions encountered
